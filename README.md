# ESTRUTURAS DE DADOS EM C

Este repositório reúne implementações didáticas das principais Estruturas de Dados
utilizando a linguagem C.

O objetivo é servir como material de estudo, consulta e prática para estudantes
e desenvolvedores que desejam compreender, de forma clara e aprofundada, o
funcionamento interno das estruturas de dados, seus algoritmos associados e a
manipulação de memória em baixo nível.

Todo o conteúdo foi organizado de forma modular, progressiva e comentada,
priorizando a compreensão dos conceitos fundamentais antes da otimização.

--------------------------------------------------

OBJETIVOS DO REPOSITÓRIO

- Apresentar as principais estruturas de dados de forma prática e didática
- Demonstrar como cada estrutura funciona internamente em linguagem C
- Explorar operações clássicas como busca, inserção e remoção
- Facilitar o estudo para disciplinas de Estruturas de Dados, Algoritmos e Programação
- Servir como referência para revisões rápidas e consultas futuras

--------------------------------------------------

ORGANIZAÇÃO E PADRÃO DO PROJETO

Cada estrutura de dados possui sua própria pasta, contendo:

- Um arquivo README explicativo sobre a estrutura
- Subpastas organizadas por tipo de operação:
  - busca
  - insercao
  - remocao
- Códigos independentes, compiláveis e comentados de forma didática

O padrão adotado permite estudar cada estrutura de forma isolada ou comparar
estratégias entre diferentes estruturas.

--------------------------------------------------

ESTRUTURAS IMPLEMENTADAS

As seguintes estruturas de dados estão presentes no repositório:

## ARRAY (VETOR)

Estrutura linear de acesso direto por índice, com foco em:
- Busca linear e busca binária
- Inserção e remoção no início, meio e fim
- Análise de custo de deslocamento de elementos

## LISTA ENCADEADA

Estrutura linear dinâmica baseada em ponteiros, abordando:
- Busca linear
- Inserções no início, meio e fim
- Remoções no início, meio e fim
- Diferenças de complexidade em relação aos arrays

## PILHA (STACK)

Estrutura do tipo LIFO (Last In, First Out), com foco em:
- Inserção no topo (push)
- Remoção do topo (pop)
- Busca sequencial desempilhando
- Uso de pilha auxiliar para preservação dos dados

## FILA (QUEUE)

Estrutura do tipo FIFO (First In, First Out), explorando:
- Inserção no fim (enqueue)
- Remoção no início (dequeue)
- Busca sequencial
- Aplicações em controle de fluxo e processamento ordenado

## ÁRVORE (TREE)

Estrutura não linear e hierárquica, incluindo:
- Busca em profundidade (DFS)
- Busca em largura (BFS)
- Busca em árvore binária de busca (BST)
- Inserção em árvores genéricas, BST e árvores balanceadas (AVL)
- Remoção de nós folha, nós com um filho e nós com dois filhos
